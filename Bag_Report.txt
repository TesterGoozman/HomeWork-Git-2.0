B-01 (Номер баг-репорта)


(Окружение)
"Postman for Mac"
Version 10.5.8
UI Version 10.5.8-ui-221206-0716
Desktop Platform Version 10.5.2
Architecture arm64
OS Platform OS X 22.1.0"

(Наименование бага)
При GET запросе компаний, с query-параметром limit=-1, в теле ответа вложены все зарегистрированные компании

(Шаги воспроизведения)
"1. Открыть Postman
2. В сайдбаре кликнуть Import
3. В открывшемся окне кликнуть Raw text
4. Вставить curl:
curl --location --request GET 'https://send-request.me/api/companies/?limit=-1'
5. Кликнуть Continue
6. В появившемся окне кликнуть import
7. Кликнуть кнопку Send"

(Ожидаемый результат)
"1. Статус-код 422;
2. В ответе JSON - присутсвует ключ details, с типом значения массив
3. В массиве есть обязательные поля loc, msg, type
4. В loc - указано, где находится ошибка. loc - является массивом
5. В msg - указано, письму с указанием на ошибку
6. В type - указан тип ошибки"

(Фактический/актуальный результат)
" 1. Статус-код 200;
 2. В response body в JSON - список всех компаний зарегистрированных на ресурсе"

(severity/серьезность(статус) ошибки)
Major

(приоритет)
Medium

(составитель баг-репорта)
Корочанский В.А.
